- a specific single day -> daily_queue_performance, period = YYYY-MM-DD
- a whole month / "this month" / "in June" -> monthly_queue_performance, period = YYYY-MM

Rules:
- Resolve relative dates/months ("today", "yesterday", "this month", "last month",
  "June", "last week") to explicit YYYY-MM-DD or YYYY-MM. Today's date is provided.
- If the user doesn't give a period, default to today (daily) or the current month
